# Sistem Informasi Pasar Terintegrasi

### Dilengkapi Dengan Modul Pembayaran untuk Pajak Harian, Pedagang, Cetak Kwitansi.
### Informasi Pedagang dan Pangkalan Minyak Tanah Sudah dilengkapi QRCode.
----------------------------------------------------

## System Requirements :
* Bahasa Utama :
  - php
  - Version 7

* Framework :
  - CodeIgniter 3

* Database :
  - Mysql (phpmyadmin)

* Library :
  - QR Code
  - Google Maps API
  - REST Controller
  - Indo Tanggal

* Template :
  - Compact
